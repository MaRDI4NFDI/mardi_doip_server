Hello, this is the MaRDI DOIP server.

It provides machine-actionable access to FAIR Digital Objects via the DOIP protocol. This endpoint supports the core DOIP operations: hello, retrieve, and invoke. So, it allows clients to check server status, resolve object metadata, stream object components, and execute registered workflows such as equation extraction. Use a DOIP-compatible client to interact with objects managed by this server on port 3567, for example the MaRDI DOIP client CLI..
